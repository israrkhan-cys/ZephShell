# ZephShell 🐚
A Unix shell built from scratch in C.

## Build & Run
```bash
git clone https://github.com/israrkhan-cys/zephshell
cd zephshell
gcc main.c -o main
./main
```

## Features
- Command execution with arguments and flags
- fork/exec/wait process model
- strtok-based argument parsing
- pipes 
- Redirection (>,>>,<)
- Multiple builtin cammand
